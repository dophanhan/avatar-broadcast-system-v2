## 🔑 密钥获取方式

### 1. 阿里云 AccessKey

1. 访问：https://ram.console.aliyun.com/manage/ak
2. 点击「创建 AccessKey」
3. 记录 `AccessKey ID` 和 `AccessKey Secret`
4. 填入环境变量 `OSS_ACCESS_KEY` 和 `OSS_SECRET_KEY`

### 2. 豆包大模型 API Key

1. 访问：https://console.volcengine.com/ark
2. 登录/注册火山引擎账号
3. 进入「密钥管理」
4. 创建 API Key
5. 填入环境变量 `DOUBAO_API_KEY`

## 📝 表格存储配置（可选）

如果后续需要云端项目存储功能，需要配置表格存储：

### 创建表格存储实例

1. 访问：https://ots.console.aliyun.com/
2. 点击「创建实例」
3. 实例名称：`avatar-v2`
4. 区域：华东 1（杭州）
5. 类型：性能型
6. 点击「立即购买」

### 创建数据表

1. 进入实例 `avatar-v2`
2. 点击「数据管理」→「创建数据表」
3. 表名：`avatar_projects`
4. 主键：
   - 名称：`project_id`
   - 类型：`字符串`
   - 自增：`是`
5. 点击「确定」

### 配置环境变量

在函数环境变量中添加：
```
OTS_ENDPOINT      = https://avatar-v2.cn-hangzhou.ots.aliyuncs.com
OTS_INSTANCE      = avatar-v2
```
